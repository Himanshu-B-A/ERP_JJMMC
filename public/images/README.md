# Images

Drop landing-page photos here. The landing page (`/`) expects these filenames —
any that exist will be used, any that are missing fall back to a soft CSS gradient
so the page never looks broken.

| Filename            | Used by                              | Suggested subject                    |
| ------------------- | ------------------------------------ | ------------------------------------ |
| `hero.jpg`          | Main hero background                 | Campus front / hospital facade       |
| `logo.png`          | Top-left logo (transparent PNG)      | JJMMC crest / logo                   |
| `campus-1.jpg`      | "Campus" showcase card 1             | Library / classroom                  |
| `campus-2.jpg`      | "Campus" showcase card 2             | Student activity / auditorium        |
| `campus-3.jpg`      | "Campus" showcase card 3             | Hospital / lab                       |
| `programme-ug.jpg`  | UG (MBBS) programme section          | Undergraduate students / classroom   |
| `programme-pg.jpg`  | PG (MD/MS) programme section         | Resident doctors / ward round        |
| `life-1.jpg`        | Student life showcase                | Sports / cultural event              |
| `life-2.jpg`        | Student life showcase                | Hostel / mess                        |
| `life-3.jpg`        | Student life showcase                | Clinical posting / OPD               |
| `cta-bg.jpg`        | Final call-to-action background      | Aerial campus / facade at dusk       |

**Tips**
- Use 1600×900 or larger landscape photos for hero/cta backgrounds.
- 1200×800 works fine for showcase cards.
- JPGs under 300 KB keep load times snappy. Use <https://tinyjpg.com> if needed.
- Transparent PNG logo at ~400×400 for `logo.png`.
